# KnittingLog-Web (Cozy Knit UI)

This repo is meant to be **public** and hosted on **GitHub Pages**.
Your actual data stays in a separate **private** repo (KnittingLog-Data).

## How to use
1. Enable GitHub Pages for this repo (Settings → Pages → Deploy from branch).
2. Open the site on your phone.
3. Tap **「同步」/「⚙︎ 雲端同步」** and fill:
   - Owner: your GitHub username
   - Repo: `KnittingLog-Data` (private data repo)
   - Branch: `main`
   - Path: `data/knitting.json`
   - Token: Fine-grained PAT with **Contents Read/Write** to the data repo
4. First press **「從 GitHub 載入」**, then **「存到 GitHub」**.

Tip: Add to Home Screen for an app-like experience.
